# Installer

Репозиторий с исходниками установщика (Inno Setup) для сборки репака Terraria.
Содержит только скрипт и ресурсы; бинарники приложения нужно положить отдельно.

## Что внутри
- `src/script.iss` — основной скрипт Inno Setup.
- `src/Terraria.ico` — иконка установщика/приложения.
- `src/WizardImageFile.bmp`, `src/WizardSmallImageFile.bmp` — картинки мастера установки.
- `src/WizardImageFile_old.bmp`, `src/WizardSmallImageFile_old.bmp` — старые версии (не используются).
- `src/logo.xnb` — ресурс логотипа TerraZ.
